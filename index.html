¿Cómo se calculan los datos?

Se descargan automáticamente las últimas tablas con reportes de la universidad de John Hopkins. A estos datos se les aplican unas correcciones en base a noticias locales debido a ligeras inconsistencias, como ser días con reportes repetidos. Luego, se extraen los últimos datos en tiempo real de Worldometer y se agregan a los reportes de John Hopkins.

¿Los datos son rigurosos?

Es importante entender que todos estos datos están basados en reportes oficiales de cada país. Nadie sabe cuántos casos reales de coronavirus hay en total, nomás pueden estimarse, por eso el interés en seguir la progresión del contagio en base a datos estadísticos, de manera similar a cómo funcionan las encuestas. Las curvas en sí son rigurosas y se revisan manualmente a diario. Sólo pueden perder algo de precisión en el último día graficado debido a la actualización en tiempo real.

¿Cómo se programó el sitio web?

Es una web estática muy sencilla basada en HTML5 y Javascript que reside en Netlify.com. Hace uso de las tecnologías Bootstrap, Font Awesome y JQuery, además de la librería Chart.js para graficar curvas. Del proceso para el cálculo de datos mencionado anteriormente resulta un archivo JSON intermedio a partir del cual se grafican las curvas y barras. Este archivo JSON se genera con un script Node.js que hace uso de las librerías D3 y Cheerio para extraer los datos de John Hopkins y Worldometer respectivamente. El código fuente del sitio es 100% público.
